# Azure OpenAI Telegram Bot

一个支持多种 AI 模型的 Telegram 机器人，基于 Azure OpenAI API。

## 🚀 支持的模型

- 🧠 **GPT-4** - 最强大的推理能力
- 🚀 **GPT-4.1** - GPT-4 升级版
- ✨ **GPT-4o** - 多模态优化模型
- ⚡ **GPT-3.5 Turbo 0125** - 快速响应
- 🤖 **Grok-3** - xAI 最新模型

## 🐳 Docker 快速部署

将.env中的bot_token替换成你的bot_token，然后运行

```bash

sudo docker compose build
docker compose up -d
